1. Clone the repo
```bash

```

2. Install dependencies
```bash
npm install
```

3. Run dev server
```bash
npm run dev
# Runs on http://localhost:5173
```

4. Build for production 
```bash
npm run build
# outputs to /dist
```
5. Preview  production build 
```bash
npm run preview
# Serves dist/ locally at http://localhost:4173
```

6. Deploy
```bash
npm run deploy
```
